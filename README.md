# passwdTester
## A password tester to increase the knowledge of how robust your password needs to be in order to be more recillient against attacks. Here you'll get a score according to how difficult the password is to crack. 

---
Passwords today has criterias as:
- Uppercase letter
- Lowercase letter
- Special characters
- Numbers
- Length 

This project is just for educational purposes and strongly advice not to write in password you use as a test to check if they are strong. The test is only to show the reasoning behind the different criterias.
